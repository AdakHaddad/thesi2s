\appendix
\chapter{Arbitrary-Fs I2S: Desain dan Penghapusan Port Legacy}
\label{app:arbitrary-fs}

Lampiran ini merangkum perubahan desain pada IP core I2S transmitter untuk menghapus port clock eksternal legacy (\texttt{audio\_44\_clk} dan \texttt{audio\_48\_clk}) sepenuhnya. Seluruh proses penyintesisan clock audio (\texttt{BCLK}, \texttt{WS}, dan \texttt{MCLK}) kini dilakukan secara murni digital menggunakan akumulator fase NCO (\textit{Numerically Controlled Oscillator}) yang berjalan pada domain clock sistem 100 MHz.

Perubahan utama pada perancangan perangkat keras:
\begin{itemize}
    \item Menghapus port input \texttt{audio\_44\_clk} dan \texttt{audio\_48\_clk} dari daftar port modul level teratas (\texttt{i2s\_dds.v}) dan modul slave AXI (\texttt{i2s\_dds\_slave\_lite\_v1\_0\_S00\_AXI.v}).
    \item Menghapus logika suppression lint untuk port clock legacy tersebut pada bagian akhir kode Verilog modul slave.
    \item Menjaga keamanan akumulator NCO dengan tetap menyertakan batas atas frekuensi (\texttt{MAX\_SAFE\_FS\_HZ}) pada modul register control untuk menghindari nilai increment yang tidak aman.
\end{itemize}

Dengan modifikasi ini, arsitektur IP core menjadi lebih sederhana, bersih, dan sepenuhnya bersifat \textit{vendor-agnostic} karena tidak lagi memerlukan clock generator analog/wizard clock tambahan untuk audio di luar clock AXI standar.
