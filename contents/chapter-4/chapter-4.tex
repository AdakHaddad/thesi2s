\chapter{Perancangan dan Implementasi}

\section{Spesifikasi dan arsitektur IP I2S TX}

Bagian ini merangkum \textbf{perancangan fungsional} IP yang menjadi inti skripsi. Implementasi difokuskan pada peripheral \textbf{I2S transmitter} dengan antarmuka \textbf{AXI4-Lite} agar dapat dikontrol oleh program C pada MicroBlaze V~\citep{xilinx_microblaze_ref_2021}.

\subsection{Daftar fitur IP}
\begin{itemize}
    \item Antarmuka \textbf{AXI4-Lite} 32 bit untuk konfigurasi dan pengisian data audio~\citep{arm_amba_axi_2010}.
    \item Mode \textbf{TX-only}, stereo, format \textbf{Philips I2S}~\citep{philips_i2s_1986}, \textbf{24 bit} per saluran dalam \textbf{slot 32 bit}.
    \item Dukungan dua keluarga frekuensi sampling nominal (\textbf{48/96 kHz} dan \textbf{44,1/88,2 kHz}), melalui pembagi clock internal dan \texttt{BUFGMUX}~\citep{xilinx_clocking_wizard_2021}.
    \item \textbf{Jalur data} lewat register \texttt{DATA\_LEFT}/\texttt{DATA\_RIGHT} dan \texttt{CONTROL}, \textbf{tanpa FIFO} internal.
\end{itemize}

\subsection{Pemetaan pin (constraint)}

Pin I2S dihubungkan ke header PMOD JA pada Basys3 melalui file \texttt{cnstr.xdc} sebagai berikut:

\begin{verbatim}
set_property -dict {PACKAGE_PIN J1 IOSTANDARD LVCMOS33} [get_ports i2s_mclk_0]
set_property -dict {PACKAGE_PIN L2 IOSTANDARD LVCMOS33} [get_ports i2s_bclk_0]
set_property -dict {PACKAGE_PIN J2 IOSTANDARD LVCMOS33} [get_ports i2s_ws_0]
set_property -dict {PACKAGE_PIN G2 IOSTANDARD LVCMOS33} [get_ports i2s_data_0]
\end{verbatim}

\subsection{Detail logika RTL (serializer dan CDC)}

\subsubsection{Mekanisme Clock Domain Crossing (CDC)}

Data audio ditulis oleh MicroBlaze V pada domain \texttt{S\_AXI\_ACLK} (100 MHz), sementara serializer beroperasi pada domain \texttt{internal\_mclk} (${\sim}24$ MHz). Sinkronisasi dilakukan menggunakan flip-flop ganda dan latching pada batas frame stereo.

\begin{lstlisting}[language=Verilog, caption={Cuplikan modul \texttt{cdc\_sync} pada RTL.}, label={lst:ch4-cdc-sync}]
always @(posedge dest_clk) begin
    stage1_q <= src_in;
    stage2_q <= stage1_q;
end
\end{lstlisting}

\subsubsection{Logika serializer Philips I2S}

\begin{lstlisting}[language=Verilog, caption={Cuplikan logika serializer I2S.}, label={lst:ch4-serializer}]
case (bit_count_q)
    6'd0, 6'd32: i2s_next_serial_bit = 1'b0; // delay bit
    default: begin
        // Serialisasi MSB first
    end
endcase
\end{lstlisting}

\section{Konfigurasi Clocking Wizard}

Clocking Wizard~\citep{xilinx_clocking_wizard_2021} dikonfigurasi menghasilkan tiga keluaran dari satu osilator 100 MHz:
\begin{enumerate}
    \item \texttt{clk\_out1} = 100 MHz (AXI ACLK).
    \item \texttt{clk\_out2} = 24,576 MHz (\texttt{audio\_48\_clk}).
    \item \texttt{clk\_out3} = 22,579 MHz (\texttt{audio\_44\_clk}).
\end{enumerate}
