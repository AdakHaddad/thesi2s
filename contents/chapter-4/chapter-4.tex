\section{Implementasi RTL pada Vivado}

Implementasi RTL (\textit{Register Transfer Level}) peripheral I2S TX dilakukan menggunakan Xilinx Vivado Design Suite. Struktur modul Verilog yang dikembangkan:

\begin{itemize}
    \item \textbf{\texttt{i2s\_axi.v}}: Modul top-level yang mengintegrasikan semua submodul.
    \item \textbf{\texttt{i2s\_axi\_slave.v}}: Antarmuka AXI4-Lite slave yang menangani transaksi baca/tulis register.
    \item \textbf{\texttt{i2s\_fifo.v}}: FIFO TX sinkron berbasis \textit{ring buffer} untuk buffer frame stereo.
    \item \textbf{\texttt{i2s\_serializer.v}}: Serializer I2S yang menghasilkan sinyal BCLK, WS, dan DATA.
    \item \textbf{\texttt{i2s\_irq.v}}: Logika pembangkitan IRQ berdasarkan level FIFO dan kondisi underrun.
\end{itemize}

Peripheral dikemas sebagai custom IP core di Vivado IP Packager dan diintegrasikan ke dalam \textit{block design} menggunakan AXI interconnect yang menghubungkannya dengan prosesor.

Aspek yang diverifikasi dalam simulasi:
\begin{itemize}
    \item Ketepatan 1-bit delay antara perubahan WS dan bit pertama data.
    \item Urutan bit MSB-first pada transmisi data 24-bit.
    \item Jumlah BCLK per kanal (32 BCLK/kanal, 64 BCLK/frame).
    \item Fungsi FIFO: push saat TX\_RIGHT ditulis, pop saat awal frame baru.
    \item Perilaku underrun: output nol saat FIFO kosong.
    \item Pembangkitan IRQ watermark saat level FIFO turun di bawah ambang batas.
\end{itemize}
