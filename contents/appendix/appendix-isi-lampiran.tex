\section{Deskripsi block design}

Vivado IP Integrator Block Design terdiri dari komponen berikut:

\begin{enumerate}
\item \textbf{MicroBlaze V (microblaze\_riscv\_0)}
\item \textbf{AXI Interconnect}
\item \textbf{Local Memory / BRAM}
\item \textbf{I2S Transmitter (custom IP i2s\_v1\_0)}
\item \textbf{AXI UART Lite}
\item \textbf{AXI Interrupt Controller} jika interrupt digunakan
\item \textbf{Clocking Wizard}
\item \textbf{Processor System Reset}
\end{enumerate}

\textbf{Koneksi utama:}
\begin{itemize}
\item Semua slave AXI4-Lite terhubung melalui AXI Interconnect.
\item Peripheral I2S menerima \texttt{audio\_clk} sebagai sumber pembangkitan \texttt{mclk/bclk/ws} dan \texttt{s00\_axi\_aclk} untuk antarmuka register AXI.
\item Sinyal eksternal utama adalah I2S: BCLK, LRCLK/WS, DATA, dan opsional MCLK.
\item UART digunakan untuk debug dan logging dari aplikasi Vitis (tidak memengaruhi laju sampel I2S).
\end{itemize}
