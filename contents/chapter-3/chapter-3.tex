\section{Arsitektur Sistem}

Sistem yang dirancang terdiri dari komponen-komponen berikut yang terintegrasi dalam Vivado IP Integrator:

\begin{enumerate}
\item \textbf{MicroBlaze} -- prosesor lunak 32-bit, \textit{clock} 100\,MHz.
\item \textbf{AXI Interconnect} -- menghubungkan MicroBlaze ke seluruh peripheral AXI4-Lite.
\item \textbf{BRAM Controller + Block RAM} -- memori program dan data (64\,KB).
\item \textbf{Clocking Wizard (MMCM)} -- menghasilkan \texttt{clk\_out1} = 100\,MHz dan \texttt{clk\_out2} $\approx$ 24,57\,MHz.
\item \textbf{Processor System Reset} -- manajemen reset tersinkronisasi.
\item \textbf{Peripheral I2S (custom IP)} -- modul yang dirancang dalam penelitian ini, terhubung via AXI4-Lite.
\item \textbf{AXI UART Lite} -- komunikasi serial untuk \textit{debugging} dan \textit{logging}.
\item \textbf{AXI Timer} -- timer untuk keperluan \textit{delay} dan pengukuran waktu.
\item \textbf{AXI Interrupt Controller} -- mengagregasi IRQ dari peripheral ke input interupsi MicroBlaze.
\end{enumerate}

\section{Perancangan Serializer I2S}

Serializer I2S bertanggung jawab menghasilkan sinyal BCLK, WS, dan DATA dari data yang di-\textit{pop} dari TX FIFO. Prinsip kerja:

\begin{enumerate}
\item \textbf{Generator BCLK}: Membagi \texttt{audio\_clk} dengan faktor $D$:
\begin{itemize}
\item Mode 1 (FS\_MODE = 1): $D = 4$ $\Rightarrow$ BCLK $\approx 6{,}142\,\text{MHz}$
\item Mode 0 (FS\_MODE = 0): $D = 8$ $\Rightarrow$ BCLK $\approx 3{,}071\,\text{MHz}$
\end{itemize}
BCLK harus memiliki \textit{duty cycle} 50\%.
\item \textbf{Generator WS}: Menghitung siklus BCLK; WS berpindah setiap 32 siklus BCLK. WS = 0 untuk 32 BCLK pertama (kanal kiri), WS = 1 untuk 32 BCLK berikutnya (kanal kanan). WS berpindah \textbf{satu siklus BCLK sebelum} bit pertama data (sesuai standar Philips).
\item \textbf{Shift Register DATA}: Pada awal setiap frame (transisi WS), data stereo di-\textit{pop} dari FIFO dan dimuatkan ke \textit{shift register} 32-bit. DATA digeser keluar MSB-\textit{first} seiring BCLK. Bit 0 (tepat setelah WS) adalah \textit{padding} nol; bit 1--24 adalah data 24-bit; bit 25--31 adalah \textit{padding} nol.
\end{enumerate}

\section{Perancangan IRQ \textit{Watermark}}

IRQ \textit{watermark} dihasilkan oleh peripheral I2S dan diteruskan ke AXI Interrupt Controller yang terhubung ke MicroBlaze. Prosedur penanganan interupsi di Vitis:

\begin{enumerate}
\item Inisialisasi AXI Interrupt Controller dan daftarkan ISR \texttt{i2s\_isr()}.
\item Isi awal FIFO dengan $\approx$200 frame (praisi sebelum ENABLE = 1).
\item Set WATERMARK = 64, aktifkan IRQ\_EN bit[0].
\item Set CONTROL bit[0] (ENABLE = 1) dan bit[4] sesuai mode (FS\_MODE).
\item Dalam ISR: baca FIFO\_LEVEL, tulis sampel hingga FIFO mendekati penuh, \textit{clear} IRQ\_STATUS bit[0].
\end{enumerate}

\section{Perancangan Aplikasi Demo Vitis (\texttt{main.cpp})}

Aplikasi demo \texttt{main.cpp} mendemonstrasikan pemutaran sinyal sinus 1\,kHz (24-bit) melalui DAC PCM5102A. Alur program:

\begin{enumerate}
\item Inisialisasi sistem dan cetak pesan via UART.
\item Bangkitkan tabel sinyal sinus 24-bit satu periode (misalnya 96 sampel untuk 1\,kHz pada 96\,kHz).
\item Inisialisasi peripheral I2S: tulis register WATERMARK, IRQ\_EN, VOLUME, CONTROL.
\item Isi awal FIFO dengan 200 frame stereo dari tabel sinyal sinus.
\item Aktifkan interupsi (ENABLE MicroBlaze interrupt, enable INTC).
\item Masuk \textit{main loop}: cetak status FIFO setiap detik; ISR menangani pengisian ulang otomatis.
\end{enumerate}

\section{Rencana Verifikasi}

Verifikasi dilakukan dalam dua tahap:

\subsection{Simulasi RTL (Vivado Simulator)}

\begin{enumerate}
\item Buat \textit{testbench} Verilog yang menstimulasi antarmuka AXI4-Lite dengan urutan tulis register (CONTROL, WATERMARK, TX\_LEFT, TX\_RIGHT berulang).
\item Amati waveform BCLK, WS, dan DATA untuk memverifikasi format I2S.
\item Verifikasi mekanisme \textit{push}/\textit{pop} FIFO dan kondisi IRQ \textit{watermark}.
\item Uji kondisi \textit{underrun} (FIFO kosong).
\end{enumerate}

\subsection{Pengujian Perangkat Keras (ILA + PCM5102A)}

\begin{enumerate}
\item Tambahkan ILA core pada desain Vivado dengan probe: \texttt{bclk}, \texttt{ws}, \texttt{data}, \texttt{fifo\_level[7:0]}, \texttt{irq}, \texttt{underrun}.
\item Sintesis dan implementasi, unduh \textit{bitstream} ke Basys3.
\item Jalankan \texttt{main.cpp} dari Vitis; buka Vivado Hardware Manager dan aktifkan ILA.
\item Ukur frekuensi BCLK dan WS dari ILA; bandingkan dengan nilai teoretis.
\item Dengarkan keluaran audio DAC PCM5102A (sinyal sinus 1\,kHz harus terdengar bersih).
\end{enumerate}
