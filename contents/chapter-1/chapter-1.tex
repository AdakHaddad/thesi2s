\section{Rumusan Masalah}

% [TODO: tulis 4–5 poin rumusan masalah sesuai panduan di atas]

Berdasarkan latar belakang yang telah diuraikan, rumusan masalah dalam
penelitian ini adalah:

\begin{enumerate}
    \item Bagaimana merancang modul peripheral I2S transmitter dalam
          Verilog dengan format audio 24-bit data dan 32-bit slot per
          channel, stereo, yang mendukung dua mode \emph{sampling rate}
          (~48 kHz dan ~96 kHz) menggunakan clock audio 24,56897 MHz yang
          dibangkitkan oleh MMCM pada Basys3?

    \item Bagaimana mengintegrasikan modul I2S transmitter tersebut sebagai
          slave AXI4-Lite ke dalam desain SoC pada FPGA Basys3 menggunakan
          Vivado IP Integrator sehingga dapat diakses oleh program C di
          Vitis melalui \emph{memory-mapped register}?

    \item Bagaimana merancang TX FIFO berkapasitas 256 frame stereo beserta
          mekanisme watermark IRQ agar CPU hanya diinterupsi saat level
          FIFO turun di bawah ambang tertentu?

    \item Bagaimana memverifikasi fungsionalitas desain secara komprehensif
          melalui simulasi RTL, pemantauan sinyal \emph{on-chip} dengan ILA,
          dan pengujian keluaran audio nyata pada DAC PCM5102A?
\end{enumerate}

\section{Tujuan Penelitian}

% [TODO: tulis 4–5 butir tujuan yang selaras dengan rumusan masalah di atas]

Tujuan penelitian ini adalah:

\begin{enumerate}
    \item Merancang modul peripheral I2S transmitter dalam Verilog yang
          mendukung format 24-bit data / 32-bit slot stereo dengan dua
          mode \emph{sampling rate} (~48 kHz dan ~96 kHz).

    \item Mengintegrasikan modul I2S transmitter sebagai slave AXI4-Lite
          ke dalam SoC pada FPGA Basys3 menggunakan Vivado IP Integrator
          dan mengembangkan program demo \texttt{main.cpp} di Vitis untuk
          mendemonstrasikan akses register dan streaming data audio.

    \item Merancang dan mengimplementasikan TX FIFO berkapasitas 256 frame
          stereo beserta mekanisme watermark IRQ dengan nilai ambang yang
          dapat dikonfigurasi.

    \item Memverifikasi fungsionalitas sistem secara end-to-end melalui
          tiga metode: (a) simulasi RTL dengan testbench Verilog,
          (b) pemantauan sinyal BCLK, WS, dan SDATA secara \emph{on-chip}
          menggunakan ILA di Vivado Hardware Manager, serta
          (c) pengujian keluaran audio nyata melalui DAC PCM5102A.
\end{enumerate}

\section{Batasan Penelitian}

% [TODO: tulis 6–8 poin batasan yang tegas dan terukur]

Penelitian ini dibatasi pada hal-hal berikut:

\begin{enumerate}
    \item Modul I2S dirancang khusus untuk mode \textbf{transmitter (TX)}
          saja; tidak mencakup mode \emph{receiver} maupun \emph{full-duplex}.

    \item Platform implementasi dibatasi pada board \textbf{Basys3}
          (FPGA Artix-7 XC7A35T-1CPG236C) dengan toolchain Xilinx
          \textbf{Vivado} dan \textbf{Vitis}.

    \item Perangkat penerima audio dibatasi pada \textbf{DAC PCM5102A};
          DAC ini beroperasi sebagai I2S slave dan tidak memerlukan
          konfigurasi melalui antarmuka lain (I2C/SPI).

    \item Format audio dibatasi pada \textbf{Philips I2S standard}:
          data 24-bit, slot 32-bit per channel, stereo (2 channel).

    \item \emph{Sampling rate} dibatasi pada dua mode:
          \textbf{Mode 0 (~47,99 kHz)} dan \textbf{Mode 1 (~95,97 kHz)},
          keduanya diturunkan dari audio clock 24,56897 MHz yang
          dibangkitkan MMCM dari osilator 100 MHz Basys3.

    \item Transfer data audio ke FIFO dilakukan melalui
          \textbf{programmed I/O} (CPU menulis register AXI4-Lite);
          tidak termasuk DMA atau burst transfer.

    \item Pengembangan software dibatasi pada lingkungan
          \textbf{bare-metal} menggunakan Vitis; tidak mencakup
          RTOS (FreeRTOS, Zephyr) maupun Linux device driver.

    \item Verifikasi daya, integritas sinyal (EMI), dan pengujian
          suhu/lingkungan tidak termasuk dalam lingkup penelitian ini.
\end{enumerate}

\noindent \textbf{Bab I} membahas latar belakang kebutuhan peripheral
I2S transmitter berbasis AXI4-Lite pada FPGA Basys3, rumusan masalah,
tujuan, batasan, dan manfaat penelitian.

\noindent \textbf{Bab III} menjelaskan perancangan sistem secara
menyeluruh: spesifikasi fungsional, diagram blok sistem, register map

watermark, dan desain program aplikasi di Vitis.

\noindent \textbf{Bab IV} memaparkan hasil implementasi di Vivado
(konfigurasi MMCM, block design, laporan sintesis/implementasi),
implementasi program Vitis, serta hasil pengujian melalui simulasi RTL,
capture ILA, dan uji audio nyata pada PCM5102A.

\noindent \textbf{Bab V} menyajikan kesimpulan yang menjawab tujuan
penelitian berdasarkan hasil pengujian, serta saran pengembangan
lanjutan (DMA, Linux driver, penambahan mode sampling, dll.).
