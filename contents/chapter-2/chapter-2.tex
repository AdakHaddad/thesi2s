% 2.2.6  FIFO dan Mekanisme Watermark Interrupt
%   Jelaskan:
%   - Konsep FIFO (First-In First-Out) sebagai buffer antara CPU dan
%     hardware yang bekerja asinkron
%   - TX FIFO 256 frame: 256 pasang sampel L/R (setiap frame = 2 × 24 bit)
%   - Pointer write (dari CPU/AXI) dan pointer read (dari serializer I2S)
%   - Kondisi FIFO: EMPTY, FULL, level counter
%   - Watermark IRQ: CPU dikonfigurasi threshold (misal 64 frame);
%     IRQ dikirim saat level FIFO ≤ threshold → CPU menulis batch baru
%   - Manfaat: mengurangi overhead CPU dibanding polling terus-menerus
